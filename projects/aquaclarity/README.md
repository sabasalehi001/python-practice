# AquaClarity

AquaClarity is a simple command-line tool for parsing and displaying mock weather data. It provides a way to view weather information for different locations and formats the output for easy readability.

## Usage

Run `python main.py` and follow the prompts to select a location and view its weather data.

## Mock Data Format

The program uses a hardcoded dictionary containing weather data for a few locations. The data includes temperature, humidity, and conditions.
