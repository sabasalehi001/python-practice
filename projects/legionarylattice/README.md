# LegionaryLattice

A simple command-line tool to convert between Roman numerals and integers (up to 3999).

## Usage

Run `python main.py` and follow the prompts.

## Examples

```
Welcome to LegionaryLattice!
Convert to Roman or Integer? (r/i): r
Enter integer (1-3999): 1984
Roman numeral: MCMLXXXIV
```

```
Welcome to LegionaryLattice!
Convert to Roman or Integer? (r/i): i
Enter Roman numeral: MCMXCIX
Integer: 1999
```
