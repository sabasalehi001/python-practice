# CogCrafter: A Simple Logic Quiz

CogCrafter is a command-line tool that presents the user with a series of logic puzzles. It tracks the user's score and provides feedback on their answers.

## Usage

1.  Clone the repository or download the `main.py` file.
2.  Run the script from your terminal: `python main.py`
3.  Answer the questions presented, following the instructions on the screen.

## Example

```
$ python main.py
Welcome to CogCrafter!

Question 1: I am always coming but never arrive. I am always present but never here. What am I?
Your answer: the future
Correct!

...
```
