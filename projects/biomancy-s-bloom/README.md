# Biomancy's Bloom

A simple command-line tool to assist with basic biological data manipulation.

## Functionality

1.  **Convert DNA to RNA:** Converts a DNA sequence to its corresponding RNA sequence.
2.  **Calculate GC Content:** Calculates the percentage of Guanine (G) and Cytosine (C) bases in a DNA sequence.

## Usage

Run `python main.py` and follow the prompts.
