# QuizzicalQuokka: Flashcard Quiz

A simple command-line flashcard quiz to help you learn new information. The quiz is driven by a set of flashcards defined directly within the script.

## Usage

1.  Run the script:

    ```bash
    python main.py
    ```

2.  Answer the questions displayed in the terminal.
3.  The quiz will provide feedback on your answers.
