# CurioCabinetCLI

A simple command-line inventory tracker for your collection of curiosities!

## Usage

Run `python main.py` to start the program.

Choose from the following options:

*   `add`: Add a new item to your cabinet.
*   `view`: View all items in your cabinet.
*   `search`: Search for an item by name.
*   `update`: Update the quantity of an existing item.
*   `exit`: Exit the program.

## Example

```
CurioCabinetCLI
Enter command (add, view, search, update, exit): add
Item name: Antique Thimble
Quantity: 1
Item added!

Enter command (add, view, search, update, exit): view
Antique Thimble: 1

Enter command (add, view, search, update, exit): exit
Goodbye!
```
