# CranialCartographer: A Flashcard Quiz

CranialCartographer is a simple command-line flashcard quiz application written in Python. It allows users to test their knowledge on a set of predefined flashcards.

## Usage

Run the `main.py` script using `python main.py` and follow the on-screen instructions.

## Flashcard Format

The flashcards are hardcoded within the `main.py` file as a dictionary where keys are questions and values are answers.  You can easily modify this dictionary to suit your specific learning needs.
