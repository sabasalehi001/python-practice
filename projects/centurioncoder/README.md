# CenturionCoder

A simple command-line tool for converting between Roman numerals and integers.

## Usage

Run `python main.py` and follow the prompts.

## Examples

```
> python main.py
Choose conversion direction:
1. Integer to Roman
2. Roman to Integer
> 1
Enter an integer (1-3999):
> 1984
MCMLXXXIV
```

```
> python main.py
Choose conversion direction:
1. Integer to Roman
2. Roman to Integer
> 2
Enter a Roman numeral:
> MCMXCIX
1999
```

## Limitations

- Only supports integers between 1 and 3999.
- Input validation is basic.
