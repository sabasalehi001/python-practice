# CipherCreek: A Simple Caesar Cipher Tool

CipherCreek is a basic command-line tool that implements a Caesar cipher for encrypting and decrypting text. It's designed for educational purposes and demonstrates a simple encryption algorithm.

## Usage

Run `python main.py` from your terminal. The program will prompt you to choose between encryption and decryption, enter your text, and specify the shift value.

## Example

```
$ python main.py
Choose an operation (encrypt/decrypt): encrypt
Enter text: Hello, World!
Enter shift value (integer): 3
Encrypted text: Khoor, Zruog!
```

```
$ python main.py
Choose an operation (encrypt/decrypt): decrypt
Enter text: Khoor, Zruog!
Enter shift value (integer): 3
Decrypted text: Hello, World!
```
