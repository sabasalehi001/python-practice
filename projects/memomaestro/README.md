# MemoMaestro

A simple command-line flashcard quiz program. Helps you test your knowledge on a set of questions and answers.

## Usage

1.  Run `python main.py`
2.  Follow the prompts to take a quiz.

## Example Flashcard Data (hardcoded in main.py):

The program comes with a pre-defined set of flashcards.  You can modify the `flashcards` variable in `main.py` to add or change flashcards.
