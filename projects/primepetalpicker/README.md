# PrimePetalPicker

A simple command-line tool to find prime numbers within a specified range. It allows you to input a start and end number and then identifies all prime numbers within that inclusive range. Input validation ensures that the start and end numbers are positive integers and that the start number is less than or equal to the end number.

## Usage

Run `python main.py` and follow the prompts to enter the start and end numbers.

## Example

```
Start number: 1
End number: 20
Prime numbers between 1 and 20: [2, 3, 5, 7, 11, 13, 17, 19]
```
