# PrimordialSieve

A simple command-line tool for finding prime numbers within a specified range.

## Usage

Run `python main.py` and follow the prompts to enter the start and end of the range.

Example:

```
python main.py
```

## Features

*   Finds all prime numbers within a user-defined range.
*   Includes basic input validation to ensure valid range limits.
*   Provides a clear and concise output of the prime numbers found.
