# ItemSleuth: A Simple Inventory Tracker

ItemSleuth is a command-line tool for tracking a small inventory of items. You can add items, view the inventory, and update quantities.

## Usage

1.  Run `python main.py`
2.  Follow the prompts to add, view, or update items.

## Example

```
$ python main.py
Welcome to ItemSleuth!

Choose an action:
1. Add Item
2. View Inventory
3. Update Quantity
4. Exit
Enter your choice (1-4): 1

Enter item name: Apple
Enter initial quantity: 10
Apple added to inventory. Quantity: 10

Choose an action:
1. Add Item
2. View Inventory
3. Update Quantity
4. Exit
Enter your choice (1-4): 2

Current Inventory:
Apple: 10

Choose an action:
1. Add Item
2. View Inventory
3. Update Quantity
4. Exit
Enter your choice (1-4): 4
Goodbye!
```
