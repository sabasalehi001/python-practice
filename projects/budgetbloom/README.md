# BudgetBloom

BudgetBloom is a simple command-line tool to help you calculate your monthly budget and savings goals.

## Usage

Run `python main.py` and follow the prompts to enter your income, expenses, and savings goals. The program will then calculate your remaining budget and progress towards your savings goal.

## Example

```
python main.py
```

## Features

*   Calculates monthly budget
*   Tracks savings progress
*   Simple and easy to use
