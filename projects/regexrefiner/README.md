# RegexRefiner

A simple command-line tool to test regular expressions against input strings.

## Usage

Run `python main.py` and follow the prompts to enter a regular expression and a test string. The program will then display whether the regex matches the string.

## Example

```
> python main.py
Enter your regular expression: hello
Enter your test string: hello world
Match found!
```

```
> python main.py
Enter your regular expression: world
Enter your test string: hello
No match found.
```
