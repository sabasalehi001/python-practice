# LexiLuminary

LexiLuminary is a simple command-line utility for manipulating text strings. It provides functionalities for highlighting specific words or phrases within a text and also for reversing the order of words in a sentence.

## Usage

Run `python main.py` and follow the prompts to choose a function and provide the necessary input.

## Features

*   **Highlight Text:** Highlights specified words or phrases within a given text using ANSI escape codes for color.
*   **Reverse Sentence:** Reverses the order of words in a sentence.
