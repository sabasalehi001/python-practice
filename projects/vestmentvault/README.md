# VestmentVault: A Simple Garment Inventory Tracker

VestmentVault is a basic command-line tool to manage your clothing inventory. You can add items, view your current inventory, and update quantities.

## Usage

Run `python main.py` to start the tracker.

## Features

*   Add new garments to your inventory.
*   View your current inventory with quantities.
*   Update the quantity of existing garments.

## Example

```
python main.py
```
