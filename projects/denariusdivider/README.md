# DenariusDivider

A simple command-line tool for exploring hypothetical currency exchange rates.

## Usage

Run `python main.py` and follow the prompts to enter the base currency, target currency, and the exchange rate.

## Example

```
Base Currency: USD
Target Currency: EUR
Exchange Rate (USD to EUR): 0.85
Amount in USD: 100

100.00 USD is equal to 85.00 EUR
```
