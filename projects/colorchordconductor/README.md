# ColorChordConductor

A simple command-line tool to convert between RGB and Hex color codes.

## Usage

Run `python main.py` and follow the prompts to choose the conversion type and input the color values.

## Example

```
$ python main.py
Choose conversion type:
1. RGB to Hex
2. Hex to RGB
Enter your choice (1 or 2): 1
Enter RGB values (e.g., 255,0,100): 255,255,255
Hex code: #FFFFFF
```

```
$ python main.py
Choose conversion type:
1. RGB to Hex
2. Hex to RGB
Enter your choice (1 or 2): 2
Enter Hex code (e.g., #FFFFFF): #000000
RGB values: (0, 0, 0)
```
