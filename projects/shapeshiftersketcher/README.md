# ShapeShifterSketcher

A simple command-line tool for generating and printing geometric patterns.

## Usage

Run `python main.py` and follow the prompts to choose a pattern and its dimensions.

## Supported Patterns

*   **Triangle:** Prints a right-angled triangle using a specified character.
*   **Square:** Prints a square using a specified character.
*   **Diamond:** Prints a diamond shape using a specified character.
