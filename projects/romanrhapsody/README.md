# RomanRhapsody

A simple command-line tool for converting between Roman numerals and integers.

## Usage

Run `python main.py` and follow the prompts to either convert from integer to Roman numeral or from Roman numeral to integer.

## Examples

```
> python main.py
Choose conversion direction:
1. Integer to Roman
2. Roman to Integer
> 1
Enter an integer (1-3999): 42
The Roman numeral is: XLII
```

```
> python main.py
Choose conversion direction:
1. Integer to Roman
2. Roman to Integer
> 2
Enter a Roman numeral: MCMXCIV
The integer value is: 1994
```
