# ChronoCrafter

A simple command-line stopwatch and countdown timer.

## Usage

Run `python main.py` and follow the prompts to start a stopwatch or countdown timer.

For the stopwatch, press Enter to start, Enter again to stop and display the elapsed time.

For the countdown timer, enter the duration in seconds. The timer will count down and then display a message.

## Example

```
python main.py
```
