# TelegraphicTranslator

A simple command-line tool for translating text to Morse code and vice versa.

## Usage

Run `python main.py` and follow the prompts to choose translation direction and input text.

## Supported Characters

*   Letters A-Z (case-insensitive)
*   Numbers 0-9
*   Space

Any other characters will be ignored.

## Morse Code Representation

*   `.` represents a dot.
*   `-` represents a dash.
*   `/` represents a space between words.
