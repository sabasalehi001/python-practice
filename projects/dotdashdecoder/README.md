# DotDashDecoder

A simple command-line Morse code translator. Translates English text to Morse code and vice-versa.

## Usage

Run `python main.py` and follow the prompts to either encode text to Morse code or decode Morse code to text.

## Example

```
$ python main.py
Choose mode (encode/decode): encode
Enter text: Hello World
.... . .-.. .-.. --- / .-- --- .-. .-.. -..
```

```
$ python main.py
Choose mode (encode/decode): decode
Enter Morse code: .... . .-.. .-.. --- / .-- --- .-. .-.. -..
HELLO WORLD
```
