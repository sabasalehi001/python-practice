# PatternPulse

PatternPulse is a simple command-line tool that generates geometric patterns based on user-defined size and a selected pattern type.

## Usage

Run `python main.py` and follow the prompts to choose a pattern type and specify the size of the pattern.

Available patterns:

*   `square`
*   `triangle`
*   `diamond`

Sizes must be positive integers.

## Examples

```
python main.py
```
