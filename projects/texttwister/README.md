# TextTwister

A simple command-line utility for manipulating strings in various ways.

## Features

*   Reverse a string.
*   Convert a string to alternating case (e.g., 'hello' becomes 'HeLlO').
*   Check if a string is a palindrome.

## Usage

```bash
python main.py reverse <string>
python main.py alternate <string>
python main.py palindrome <string>
```

## Examples

```bash
python main.py reverse hello
# Output: olleh

python main.py alternate hello
# Output: HeLlO

python main.py palindrome racecar
# Output: True

python main.py palindrome hello
# Output: False
```
