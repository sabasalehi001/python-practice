# BudgetBuddyCLI

A simple command-line tool to help beginners manage their personal finances by estimating savings based on income and expenses, and calculating debt payoff time.

## Usage

Run `python main.py` and follow the prompts. The tool provides two main functions:

1.  **Savings Estimator:** Calculate potential savings based on income and expenses.
2.  **Debt Payoff Calculator:** Estimate the time required to pay off a debt.

## Example

```
python main.py
```
