# GeneScribeLite

A simple command-line tool to help with basic DNA sequence manipulation and analysis.

## Features

*   **Reverse Complement:** Calculates the reverse complement of a DNA sequence.
*   **GC Content:** Determines the GC content percentage of a DNA sequence.

## Usage

```bash
python main.py reverse_complement ATGC
python main.py gc_content ATGC
```

## Input

DNA sequences should only contain the characters A, T, G, and C.
