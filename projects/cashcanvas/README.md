# CashCanvas: Simple Personal Budget Calculator

CashCanvas is a beginner-friendly Python program that helps you calculate your remaining budget after entering your income and expenses. It provides a simple command-line interface for inputting your financial information and displays the calculated budget.

## Usage

1.  Clone the repository or download the `main.py` file.
2.  Run the program from your terminal using `python main.py`.
3.  Follow the prompts to enter your income and expenses.
4.  The program will then calculate and display your remaining budget.

## Example

```
python main.py
Enter your monthly income: 3000
Enter an expense (or type 'done'): Rent
Enter the amount: 1000
Enter an expense (or type 'done'): Groceries
Enter the amount: 300
Enter an expense (or type 'done'): Entertainment
Enter the amount: 200
Enter an expense (or type 'done'): done

Your remaining budget: $1500.00
```
