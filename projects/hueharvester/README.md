# HueHarvester

A simple command-line tool for converting color codes between hexadecimal and RGB formats.

## Usage

Run `python main.py` and follow the prompts to select the conversion type and input the color code.

## Example

```
$ python main.py
Choose conversion type:
1. Hex to RGB
2. RGB to Hex
Enter your choice (1 or 2): 1
Enter hex color code (e.g., #FFFFFF): #FF0000
RGB: (255, 0, 0)
```

```
$ python main.py
Choose conversion type:
1. Hex to RGB
2. RGB to Hex
Enter your choice (1 or 2): 2
Enter RGB values (e.g., 255,0,0): 0,0,255
Hex: #0000ff
```
