# TablatureTapestry

TablatureTapestry is a simple command-line tool that helps you create Markdown tables from user-provided data.

## Usage

Run `python main.py` and follow the prompts to enter the table header and data rows. The program will then output the Markdown table to the console.

## Example

```
python main.py
```

Follow the prompts, and you'll get a Markdown table like this:

```
| Header 1 | Header 2 |
|---|---|
| Data 1   | Data 2   |
| Data 3   | Data 4   |
```
