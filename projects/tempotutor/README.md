# TempoTutor

A simple command-line tool to help you practice consistent timing.

## Usage

1.  Clone the repository or download the `main.py` file.
2.  Run the script with `python main.py`.
3.  Follow the on-screen prompts to start a timed interval practice session.

## Features

*   Set a specific interval duration (in seconds).
*   Practice hitting a key at regular intervals.
*   Get feedback on your timing accuracy.
