# PixelWeaver

PixelWeaver is a simple command-line tool that generates geometric patterns using characters.  It allows users to specify a pattern type (square, triangle), a character to use for drawing, and the size of the pattern. The program then prints the corresponding pattern to the console.

## Usage

Run `python main.py` and follow the prompts to select a pattern type, enter a character, and specify the size of the pattern. The pattern will then be printed to your terminal.

## Examples

```
$ python main.py
Pattern type (square or triangle): square
Character: *
Size: 5

*****
*****
*****
*****
*****
```

```
$ python main.py
Pattern type (square or triangle): triangle
Character: #
Size: 4

#
##
###
####
```
