# FlorinFlux: Currency Conversion Calculator

FlorinFlux is a simple command-line tool that allows you to convert between two currencies using predefined exchange rates.

## Usage

Run `python main.py` and follow the prompts to enter the currencies and amount you want to convert.

## Example

```
$ python main.py
Base Currency (USD, EUR, GBP): USD
Target Currency (USD, EUR, GBP): EUR
Amount to convert: 100
100.00 USD is equal to 92.00 EUR
```

## Supported Currencies

*   USD (United States Dollar)
*   EUR (Euro)
*   GBP (British Pound)
