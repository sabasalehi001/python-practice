# PrimePrism: A Simple Prime Number Explorer

PrimePrism is a command-line tool that allows you to explore prime numbers within a specified range.  It provides a basic way to identify and display prime numbers.

## Usage

Run the script from your terminal:

```bash
python main.py
```

You will be prompted to enter a starting and ending number for the range you want to explore. The program will then display all prime numbers within that range.

## Example

```
Enter the starting number: 1
Enter the ending number: 20

Prime numbers between 1 and 20:
2
3
5
7
11
13
17
19
```

## Functions

*   `is_prime(number)`: Checks if a given number is prime.
*   `find_primes(start, end)`: Finds all prime numbers within a specified range.
