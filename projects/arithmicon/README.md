# Arithmicon

A simple command-line tool to solve basic arithmetic puzzles.

## Usage

Run `python main.py` and follow the prompts to select a puzzle type and provide the necessary input.

## Available Puzzles

1.  **Missing Number:** Find the missing number in a sequence (e.g., 2, 4, ?, 8, 10).
2.  **Target Sum:** Find two numbers in a list that add up to a target sum.
