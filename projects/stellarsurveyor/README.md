# StellarSurveyor

A simple ASCII game where you explore a procedurally-generated galaxy.

## How to Play

1.  Run `python main.py`.
2.  Follow the prompts to navigate your spaceship and survey the stars.
3.  Use the compass directions (N, S, E, W) to move.
4.  Discover planets with varying resources.
5.  The game ends when you choose to quit or run out of fuel.

## Controls

*   `N`: Move North
*   `S`: Move South
*   `E`: Move East
*   `W`: Move West
*   `Q`: Quit

## Note
This is a very basic implementation, focusing on core gameplay loop and simple ASCII graphics. Have fun exploring!
