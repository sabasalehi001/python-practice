# ChronoSculptor

A simple command-line tool to sculpt time intervals.  It allows you to start, stop, and display elapsed time.

## Usage

1.  Run `python main.py`
2.  Follow the on-screen prompts to start, stop, and view the elapsed time.

## Functionality

*   Start the timer.
*   Stop the timer.
*   View the elapsed time since the timer started.
*   Clear the elapsed time.
