# PennyPincherProjections

A simple command-line tool to project savings based on recurring deposits and interest rates.

## Usage

Run `python main.py` and follow the prompts to enter your initial deposit, monthly deposit, annual interest rate, and the number of years you plan to save.

The program will output the projected total savings after the specified number of years.

## Example

```
Initial Deposit: 1000
Monthly Deposit: 100
Annual Interest Rate (%): 5
Number of Years: 10

Projected Savings: $16727.79
```
