# TrinketTracer: A Simple CLI Inventory Tracker

TrinketTracer is a basic command-line tool for tracking a small collection of items (trinkets). You can add, view, and update the quantity of your trinkets.

## Usage

Run the script: `python main.py`

Follow the on-screen prompts to manage your trinkets.

## Example

```
Welcome to TrinketTracer!

Options:
1. Add Trinket
2. View Trinkets
3. Update Trinket Quantity
4. Exit

Enter your choice (1-4): 1

Enter trinket name: Shiny Rock
Enter initial quantity: 3
Trinket added!
```
