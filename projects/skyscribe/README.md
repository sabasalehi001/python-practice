# SkyScribe: Mock Weather Data Parser

SkyScribe is a simple command-line tool that parses mock weather data and displays it in a user-friendly format.

## Usage

Run `python main.py` and follow the on-screen prompts.

## Example Weather Data Format

The program expects weather data in the following format:

```
City:London,Temperature:15,Condition:Cloudy
City:Tokyo,Temperature:22,Condition:Sunny
```

## Functionality

*   Parses weather data from a user-provided string.
*   Displays weather information for each city.
*   Provides basic input validation.
