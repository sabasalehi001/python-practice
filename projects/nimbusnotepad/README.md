# NimbusNotepad

NimbusNotepad is a simple command-line tool for parsing and displaying mock weather data.

## Usage

Run the script `main.py` and follow the prompts to view weather data for a specific location and date.

## Example Data Format

The script expects weather data in a specific format:

```
location,date,temperature,condition
```

For example:

```
New York,2024-01-15,5,-3,Cloudy
```

## Note

This project uses mock weather data. No external weather services are accessed.
