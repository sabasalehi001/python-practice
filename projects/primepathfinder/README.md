# PrimePathfinder

A simple command-line tool to find prime numbers within a given range.

## Usage

Run `python main.py` and follow the prompts to enter the start and end of the range.

## Example

```
Start of range: 1
End of range: 20
Prime numbers between 1 and 20: [2, 3, 5, 7, 11, 13, 17, 19]
```
