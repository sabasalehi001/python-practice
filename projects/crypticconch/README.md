# CrypticConch

A simple command-line tool for basic text encryption and decryption using a Caesar cipher with a user-defined shift value.

## Usage

Run `python main.py` and follow the prompts.

You can choose to encrypt or decrypt text, and you'll be asked to enter the text and the shift value (an integer).

## Example

```
$ python main.py
Choose an operation (encrypt/decrypt): encrypt
Enter the text: Hello, world!
Enter the shift value: 3
Ciphertext: Khoor, zruog!
```
