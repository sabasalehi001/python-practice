# RuneCipher

RuneCipher is a simple command-line tool for basic text encryption and decryption using a substitution cipher based on a fictional rune alphabet.

## Usage

```bash
python main.py encrypt <text>
python main.py decrypt <text>
```

Where `<text>` is the string you want to encrypt or decrypt.

## Example

```bash
python main.py encrypt hello
```

```bash
python main.py decrypt ᚺᛂᛚᛚᛟ
```
